# mdr

A markdown renderer.

## Usage

```
mdr 0.1.0

USAGE:
    mdr [OPTIONS] <file>

OPTIONS:
    -i, --ip <ip>        The ip to serve the file from [default: 127.0.0.1]
    -p, --port <port>    The port to serve the file from [default: 8080]

ARGS:
    <file>    The path to the markdown file to render
```
